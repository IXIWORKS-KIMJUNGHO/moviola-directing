# Canonical Directing Rules

Use skill version `1`. Report this exact value in every `get_draft_outline` call.

## Cut splitting flow

컷 분할 흐름은 Establishing → Development → Emphasis → Resolution이다. 연속 동작은 한 컷으로 유지하고, 분위기를 유지하는 순간은 길게 두며, 대사 교차 편집은 2–3회 이내로 제한한다.

## Shot grammar

- `shotSize × lensType`: Wide 렌즈는 Wide/EST shot과, Tele/Macro는 CU/ECU와 물리적으로 호환되어야 한다.
- `shotSize × cameraAngle`: 의도가 정합해야 하며 CU에 Bird's Eye를 사용하지 않는다.
- Establishing Shot은 새 location, 시간 점프, 외부·대규모 공간, 감정 리셋 중 하나 이상을 충족할 때만 사용한다. Scene당 최대 한 번, `cutIndex=0`을 우선한다. 조건이 없으면 plot beat에 맞는 Full, Long, Group, Medium 등을 선택한다.

The rules are satisfied when each authored or reviewed Cut has a deliberate dramatic function, physically coherent shot grammar, and continuity with adjacent Cuts.
