# Review

- Use `get_scene_board` to inspect composition, coverage, shot progression, and rhythm across a Scene. Use `get_cut_image` only when facial or visual detail requires the original Cut image.
- Compare the work with the North Star, Decision Memo, selected genre guide, Character continuity, spatial axis, and the establishing → development → emphasis → resolution flow.
- Treat the result as the terminal model's self-review. A separate MOVIOLA critic did not review it.
- When the director requests only an opinion, return the diagnosis without mutation.
- When the director requests alternatives, propose specific Cut-level before/after changes and wait for a selection before paid rerendering.
- Keep accepted decisions stable. Reopen a recorded rejected approach only when the director asks.

The review is complete when every finding points to visible evidence or returned Draft state, and advice is clearly separated from any executed edit.
